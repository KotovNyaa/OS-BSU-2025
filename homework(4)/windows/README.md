
# killer dummy

Реализация  с использованием библиотеки `<windows.h>` для Windows.

## Сборка и запуск
Для управления проектом используется не Makefile а через консоль. Находясь в корневой папке, выполните:

*    `cl /EHsc dummy.cpp
      cl /EHsc killer.cpp
      cl /EHsc user.cpp`   — собрать проект.

*     `.\user.exe`  - запуск проекта

## Описание
В ходе лабораторной работы было реализовано dummy чтобы его килять, киллер чтобы килять dummy, и user чтобы запускать киллер который киляет dummy
