
# killer dummy

Реализация  с использованием библиотеки `<unistd.h>` для Linux.

## Сборка и запуск
Для управления проектом используется не Makefile. Находясь в корневой папке, выполните:

*    `g++ -std=c++17 dummy.cpp -o dummy -pthread
      g++ -std=c++17 killer.cpp -o killer
      g++ -std=c++17 user.cpp -o user -pthread`   — собрать проект.

*     `./user`  - запуск проекта

## Описание
В ходе лабораторной работы было реализовано dummy чтобы его килять, киллер чтобы килять dummy, и user чтобы запускать киллер который киляет dummy
