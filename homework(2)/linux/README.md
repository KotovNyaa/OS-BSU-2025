# Matrix Multiplication (POSIX Threads)

Реализация многопоточного умножения матриц с использованием библиотеки `pthread` для Linux.

## Сборка и запуск
Для управления проектом используется Makefile. Находясь в корневой папке, выполните:

*   `make build`   — собрать проект.
*   `make run`     — собрать и запустить программу.
*   `make clean`   — очистить скомпилированные файлы.
*   `make rebuild` — полная пересборка проекта.

## Описание
В ходе лабораторной работы было реализовано сравнение двух алгоритмов:

1.  **Standard Algorithm** — обычное умножение по определению (однопоточное).
2.  **Parallel Block Multiplication (pthread)** — многопоточное умножение с разбивкой на блоки и транспонированием матрицы B для оптимизации кэша.
