# Matrix Multiplication (WinAPI)

Реализация многопоточного умножения матриц с использованием нативного API Windows (`<windows.h>`).

## Сборка и запуск
Для управления проектом используется Windows, а для запуска использовалась Visual Studio:

*   `нажать на собрать проект`   — собрать проект.
*   `нажать на собрать и запустить программу`     — собрать и запустить программу.
*   `нажать на очистить скомпилированные файлы`   — очистить скомпилированные файлы.
*   `нажать на полная пересбока проекта` — полная пересборка проекта.

## Описание
В ходе лабораторной работы было реализовано сравнение двух алгоритмов:

1.  **Standard Algorithm** — обычное умножение по определению (однопоточное).
2.  **Parallel Block Multiplication (WinAPI)** — многопоточное умножение с разбивкой на блоки и транспонированием матрицы B для оптимизации кэша.
