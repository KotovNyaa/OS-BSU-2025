# Домашняя работа №2:

Репозиторий содержит реализацию алгоритма блочного умножения двух матриц ($A \times B = C$) с использованием многопоточности.

Проект реализован для двух платформ: **Windows** и **Linux**. Реализация выполнена с использованием трех различных библиотек для работы с потоками, как того требует задание:
1.  `#include <thread>` (Стандартная библиотека C++)
2.  `#include <pthread>` (POSIX Threads)
3.  `#include <windows.h>` (Windows API — только для Windows)

Для анализа производительности программа подсчитывает время работы для различного количества блоков (и, соответственно, потоков) и сравнивает его с обычным однопоточным алгоритмом.

## Структура проекта

*   `windows/` — Решение Visual Studio 2022.
*   `linux/` — Проект на базе CMake для линукса.
*   `multiplatform/` — Проект на базе CMake но мультиплатформный
