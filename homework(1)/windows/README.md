# Первая домашняя работа по OS

## Библиотеки

### `numberLIB` (статическая)

-   Предоставляет класс `Number` для работы с числами с плавающей запятой.
-   Перегружены основные арифметические операторы: `+`, `-`, `*`, `/`.
-   Содержит статические методы для получения констант нуля (`Number::Zero()`) и единицы (`Number::One()`).

### `vectorLIB` (динамическая)

-   Предоставляет класс `Vector` для работы с 2D-векторами в декартовых координатах.
-   Реализует операцию сложения векторов.
-   Содержит методы для расчёта полярных координат (радиус и угол).
-   Полностью зависит от `numberLIB` для всех числовых операций.
-   Содержит статические методы для получения нулевого вектора (`Vector::Zero()`) и вектора `(1, 1)` (`Vector::One()`).

## Сборка и запуск в Visual Studio 2022

### Запуск

-   **Visual Studio 2022** с установленным набором инструментов **"Разработка классических приложений на C++"**.

1.  **Откройте решение.**
    -   Запустите Visual Studio 2022.
    -   Откройте файл решения (`.sln`), который находится в корневой директории проекта.

2.  **Назначьте запускаемый проект.**
    -   В **Обозревателе решений** (Solution Explorer) найдите проект `Main`.
    -   Щелкните по нему правой кнопкой мыши и выберите **"Назначить в качестве запускаемого проекта"** (Set as StartUp Project).

3.  **Соберите и запустите проект.**
    -   Нажмите клавишу **`Ctrl+F5`** (Запуск без отладки) или **`F5`** (Запуск с отладкой).
    -   Visual Studio автоматически соберет все зависимости (`numberLIB`, `vectorLIB`) и запустит исполняемый файл.

### Ожидаемый вывод

При успешном запуске вы увидите в консоли следующий результат:

```text

b = 3
Number::Zero() = 0
Number::One() = 1
a + b = 18.5
a - b = 12.5
a * b = 46.5
a / b = 5.16667

--- Демонстрация библиотеки Vector ---
Вектор v1 = (3, 4)
Вектор v2 = (1, 2)
Vector::Zero() = (0, 0)
Vector::One() = (1, 1)
Сумма v1 + v2 = (4, 6)
Для вектора v1:
  - Полярный радиус (длина): 5
  - Полярный угол (в радианах): 0.927295
```
