# Домашняя работа №1:

Репозиторий содержит реализацию проекта, состоящего из статической (`numberLIB`) и динамической (`vectorLIB`) библиотек.

Проект реализован для двух платформ: **Windows** и **Linux**. Исходный код классов `Number` и `Vector` полностью идентичен для обеих платформ, различаются только системы сборки.

## Структура проекта

*   `windows/` — Решение Visual Studio 2022.
*   `linux/` — Проект на базе CMake.

## Сборка и запуск

### Windows

Для сборки используется Visual Studio. Откройте файл решения `windows/Libraries.sln` и следуйте инструкции в файле [windows/README.md](./windows/README.md).

### Linux

Для сборки используется CMake. Подробные инструкции по сборке и запуску находятся в файле [linux/README.md](./linux/README.md).
